# Hexo 的安装与初始化

  1. 在博客目录下右键 git bash here ，执行 npm install hexo-cli -g 安装 Hexo

  2. hexo init 初始化，自动配置文件与下载相关依赖

  3. hexo g 从 md 文件生成 html

  4. hexo s 启动本地服务

  5. 访问 localhost:4000

# 部署到 Github

  1. ssh-keygen -t rsa [-C comment] 生成密钥，三下回车

  2. 将 C:/Users/<username>/.ssh/id_rsa.pub 的内容复制到 GitHub -> Settings -> SSH and GPG keys -> New SSH keys

  3. ssh -T git@github.com 验证，输入 yes

  4. 创建 Git 仓库 <git username>.github.io ，其余选项默认

  5. 在 BlogRoot/_config.yml 中修改以下内容

  6. npm install hexo-deployer-git --save 安装 git 插件

  7. hexo clean 清理、 hexo g 生成、 hexo d 部署

  8. 访问 <git username>.github.io ，若 404 则稍等一会，或重复上一步
