Hexo的安装与部署

安装 Hexo

  1. npm install hexo-cli -g


![image-20220425204530718](image-20220425204530718.png)

  2. hexo g 


![image-20220425211807237](image-20220425211807237.png)

  3. 访问 localhost:4000

部署到Github

  1. ssh-keygen -t rsa [-C comment] #三下回车


![image-20220425224815315](image-20220425224815315.png)

  2. 创建Git仓库 <git username>.github.io，其余选项默认

  3. 打开BlogRoot/_config.yml修改以下内容

  4. npm install hexo-deployer-git --save


<img src="hexod.png" alt="hexod" style="zoom:80%;" />

  5. 访问 <git username>.github.io，若404则稍等一会，或重复上一步
