Install Notes

Node.js – Install

  1. https://nodejs.org/en/

  2. 一路next，“Automatically”选项勾选的话会先安装chocolatey，然后用choco安装最新版python和VS-workload-vctools，建议不勾

  3. 修改npm的cache与全局路径（先新建相应文件夹）

  4. 环境变量

    1. 将用户变量下”Roaming\npm”修改为”nodejs\node_global”
    2. 添加系统变量NODE_PATH: “F:\nodejs\node_global\node_modules”（文件夹不存在也没关系）

  5. “F:\nodejs”修改User权限为完全控制

  6. 验证

Java – Uninstall

  1. 控制面板卸载Java与Java SE
  2. 删除相关环境变量：JAVA_HOME, CLASSPATH, PATH下的Java路径
  3. 清除注册表：HKEY_CURRENT_USER\Software\JavaSoft
