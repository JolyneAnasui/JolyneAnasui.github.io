Hexo的安装与部署

Hexo的安装与初始化

  1. 在博客目录下右键git bash here，执行npm install hexo-cli -g安装Hexo

  2. hexo init初始化，自动配置文件与下载相关依赖

  3. hexo g从md文件生成html

  4. hexo s启动本地服务

  5. 访问 localhost:4000

部署到Github

  1. ssh-keygen -t rsa [-C comment]生成密钥，三下回车

  2. 将C:/Users/<username>/.ssh/id_rsa.pub的内容复制到GitHub -> Settings -> SSH and GPG keys -> New SSH keys

  3. ssh -T git@github.com验证，输入yes

  4. 创建Git仓库<git username>.github.io，其余选项默认

  5. 在BlogRoot/_config.yml中修改以下内容

  6. npm install hexo-deployer-git --save安装git插件

  7. hexo clean清理、hexo g生成、hexo d部署

  8. 访问 <git username>.github.io，若404则稍等一会，或重复上一步

Install Notes

Node.js – Install

  1. https://nodejs.org/en/

  2. 一路next，“Automatically”选项勾选的话会先安装chocolatey，然后用choco安装最新版python和VS-workload-vctools，建议不勾

  3. 修改npm的全局路径与cache路径（先新建相应文件夹）

  4. 环境变量

    1. 将用户变量下Roaming\npm修改为nodejs\node_global
    2. 添加系统变量NODE_PATH: F:\nodejs\node_global\node_modules（文件夹不存在也没关系）

  5. 文件夹F:\nodejs修改User权限为完全控制

  6. 验证

Java – Uninstall

  1. 控制面板卸载Java与Java SE
  2. 删除相关环境变量：JAVA_HOME, CLASSPATH, PATH下的Java路径
  3. 清除注册表：HKEY_CURRENT_USER\Software\JavaSoft
