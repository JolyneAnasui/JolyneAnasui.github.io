Hexo+Typora图片路径问题

  1. npm install --save https://github.com/CodeFalling/hexo-asset-image安装插件 ，忽略vulnerabilities

  2. 在BlogRoot/_config.yml中设置post_asset_folder: true

  3. 设置Typora图片存储路径，hexo生成时无需删除同名文件夹前缀路径

  4. 删除图片名字或alt属性，否则图片下方会有标注文字
